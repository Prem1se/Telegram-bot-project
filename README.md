# Анализ данных

## Основной вывод исследования:

> Данные демонстрируют, что значительная часть игроков сталкивается с трудностями во взаимодействии с сокомадниками. 
Проблемы носят комплексный характер: от существенного разброса в рейтинге и опыте игроков внутри команд до серьёзных коммуникационных сложностей, включая проявления токсичности.
